# auto-disenchanter-v2

League of Legends tool to open loots, disenchant, redeem loots, buy champions in batch for multiple accounts.
![Demo](https://raw.githubusercontent.com/sandbox-pokhara/auto-disenchanter-v2/master/images/demo.png)

## Usage

### Supported input csv format

```
username1:password1
username2:password2
username3:password3
```
